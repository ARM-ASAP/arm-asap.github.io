ESP32P4 : 3D ENGINE

3D vectors can be challenging to analyze, particularly when dealing with complex operations and relationships. To simplify the process and better understand the impact of these operations, I developed a lightweight 3D engine for visualization. All operations are performed by the CPU, with no hardware acceleration, making it platform-independent. However, the design is flexible, allowing for easy integration of specific hardware acceleration when needed.

FLOOR HEATING SYSTEM

Complete floor heating system. Xiaomi mijia 2 BLE themrometer/higrometer (based on telink TLSR8258 MCU) was flashed with custom software. Base unit (ESP32S3) is reading BLE data from Mijia and controls electric thermal actuators. Setup up of the base unit is done via UI on touch screen (ST7703 for LCD + GT911 for touch) running on ESP32-P4 + ESP32-C6 HOSTED for connectivity.
